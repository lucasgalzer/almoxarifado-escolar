# Almoxarifado Escolar

Sistema web para controle de estoque, empréstimos e solicitações internas de materiais escolares.

## Tecnologias

- **Frontend:** React + Vite
- **Backend:** Node.js + Express
- **Banco de dados:** PostgreSQL

## Como rodar localmente

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm run dev
